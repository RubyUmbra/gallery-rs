# gallery-rs

Image sorting application.

## Usage

`gallery-rs <path>`

`<path>` -- path of directory with pictures to sort

## Flags

`-h, --help` -- Print help information

`-V, --version` -- Print version information

## Key bindings

`Esc`, `q` -- quit

`<-` -- previous picture

`->` -- next picture

`Space` -- move current picture to `<path>/del`
